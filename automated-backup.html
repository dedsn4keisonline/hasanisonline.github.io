Overview

Losing important data is a nightmare, so I built an automated backup system to ensure my files are always safe. This project uses Rsync for Linux and Task Scheduler for Windows to create regular backups of critical data. Let me walk you through how I set it up!

What I Built

Here are the key components of my automated backup system:

  • Cross-Platform Support: Works on both Linux and Windows systems.
  • Incremental Backups: Uses Rsync to only back up changes, saving time and storage.
  • Scheduled Backups: Automates backups using Cron (Linux) and Task Scheduler (Windows).
  • Encryption: Secures backups with AES-256 encryption.
